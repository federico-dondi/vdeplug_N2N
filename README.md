# vdeplug_N2N
Peer-to-Peer VPN plugin module for [VDEPlug](https://github.com/rd235/vdeplug4). Based on the [N2N](https://github.com/ntop/n2n) library.
## How to install:

1. `git clone` this repo
2. `cd` into the new directory
3. `mkdir build`
4. `cd build`
5. `cmake ..`
6. `make configure `
7. `sudo make install`

## Usage:

Run:
`sudo vde_plug n2nEdge://`
